پروژه BehradB2-AutoScientist یک سیستم هوش مصنوعی کاملاً خودمختار است که در نقش یک محقق و مهندس نرم‌افزار ارشد عمل می‌کند. این سیستم به گونه‌ای طراحی شده است که بدون نیاز به هیچ‌گونه دخالت یا نظارت انسانی، ایده‌پردازی کرده، معماری پروژه‌های پایتون را طراحی نموده، کدهای کامل و کاربردی نوشته و در نهایت مستندات دقیق پروژه را تولید کند. با بهره‌گیری از یک جریان کار چند-عاملی (Multi-Agent) پیشرفته و ادغام یکپارچه با روترهای هوشمند محلی مانند 9Router، این سیستم می‌تواند چرخه‌های متوالی و بی‌وقفه تولید کد را اجرا کرده و محدودیت‌های رایج API را به طور کامل دور بزند.

## 🏗️ معماری هسته سیستم
این سیستم بر پایه یک معماری سه‌گانه متشکل از عامل‌های هوشمند است که توسط یک مکانیزم حلقه اصلی کنترل می‌شوند:

* عامل اول (موتور ایده‌پردازی): به مدل زبانی دستور می‌دهد تا یک اسکریپت پایتون جدید و کاربردی برای دنیای واقعی اختراع کند. این عامل خروجی‌ها را مجبور به رعایت ساختار دقیق JSON می‌کند و از عبارات باقاعده (Regex) برای استخراج امن معماری پروژه استفاده می‌کند.
* عامل دوم (برنامه‌نویس خودمختار): ابزار قدرتمند aider را از طریق پردازش‌های پس‌زمینه مدیریت می‌کند. این عامل به طور خودکار مخازن Git را مقداردهی اولیه کرده، تمام پرسش‌های تعاملی ترمینال را به صورت خودکار تایید می‌کند (yes-always) و کدها را بر اساس نقشه استخراج شده می‌نویسد.
* عامل سوم (موتور مستندسازی): پروژه را با تولید فایل‌های مستندات Markdown و یک فایل run_metadata.json حاوی متادیتا و تایم‌استمپ دقیق ISO برای حسابرسی نهایی تکمیل می‌کند.

## 🌐 آموزش جامع و نحوه کارکرد ناین روتر (9Router)
برای اینکه BehradB2 بتواند در حلقه‌های مداوم کار کند و بودجه یا توکن‌های API شما به سرعت تمام نشود، این پروژه به شدت برای کار با 9Router بهینه‌سازی شده است.

### ناین روتر چیست؟
ناین روتر یک پروکسی و دروازه (Gateway) هوشمند، رایگان و اوپن‌سورس است که روی سیستم محلی شما اجرا می‌شود. این ابزار به جای اینکه درخواست‌های هوش مصنوعی شما را مستقیماً به یک شرکت (مثل OpenAI) بفرستد، آن‌ها را دریافت کرده و به صورت هوشمند بین بیش از ۴۰ ارائه‌دهنده مختلف توزیع می‌کند.

### چرا استفاده از آن در این پروژه ضروری است؟
* دور زدن محدودیت‌ها (Rate Limits): ناین روتر از یک سیستم آبشاری (Fallback) استفاده می‌کند. اگر سهمیه اکانت اصلی شما تمام شود، بدون توقف پروژه، درخواست را به سرویس‌های ارزان‌تر و در نهایت به سرویس‌های رایگان می‌فرستد.
* فشرده‌سازی توکن‌ها (RTK Compression): ابزارهایی مثل عامل کدنویس ما برای بررسی کدها از git diff استفاده می‌کنند که توکن‌های زیادی مصرف می‌کند. سیستم RTK در ناین روتر، فضاهای خالی و کدهای اضافی را قبل از ارسال به هوش مصنوعی حذف می‌کند و بین ۲۰ تا ۴۰ درصد در مصرف توکن‌های ورودی صرفه‌جویی می‌کند.
* حالت انسان اولیه (Caveman Mode): این ویژگی باعث می‌شود هوش مصنوعی از زدن حرف‌های محاوره‌ای اضافه پرهیز کند و فقط کد خالص را برگرداند که توکن‌های خروجی را تا ۶۵٪ کاهش می‌دهد.

## 🛠️ راهنمای نصب و راه‌اندازی
اطمینان حاصل کنید که پایتون نسخه 3.9 یا بالاتر بر روی سرور یا سیستم شما نصب شده است.

۱. مخزن را کلون کرده و وارد پوشه پروژه شوید:
git clone https://github.com/BehradB289/BehradB2-AutoScientist.git
cd BehradB2-AutoScientist

۲. یک محیط مجازی پایتون (Virtual Environment) ایجاد و فعال کنید:
python -m venv venv
source venv/bin/activate  # در ویندوز: venv\Scripts\activate

۳. کتابخانه‌های مورد نیاز را نصب کنید:
pip install -r requirements.txt

## 🚀 نحوه استفاده و چرخه خودمختار

۱. ابتدا سرور روتر کامپایلر را اجرا کنید:
python compiler_server.py
